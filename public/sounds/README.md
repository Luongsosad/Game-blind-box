# Hướng dẫn thêm file âm thanh

Để trò chơi có âm thanh, bạn cần thêm các file âm thanh vào thư mục `public/sounds/`:

## File cần thiết:

1. **background.mp3** - Nhạc nền cho trò chơi
2. **correct.mp3** - Âm thanh khi trả lời đúng
3. **wrong.mp3** - Âm thanh khi trả lời sai

## Cách tải file âm thanh miễn phí:

### Tùy chọn 1: Freesound.org
- Truy cập: https://freesound.org/
- Tìm kiếm: "correct answer", "wrong answer", "game background music"
- Tải về và đổi tên file

### Tùy chọn 2: Mixkit.co
- Truy cập: https://mixkit.co/free-sound-effects/
- Chọn "Game" hoặc "Success/Error" sounds
- Tải về miễn phí

### Tùy chọn 3: Pixabay
- Truy cập: https://pixabay.com/sound-effects/
- Tìm kiếm âm thanh phù hợp

### Tùy chọn 4: Tự tạo (Tạm thời)
Nếu chưa có file âm thanh, hệ thống vẫn hoạt động bình thường (không có lỗi).
Audio sẽ tự động phát khi bạn thêm file vào.

## Gợi ý từ khóa tìm kiếm:
- **Correct**: "success", "win", "correct", "ding", "bell"
- **Wrong**: "error", "fail", "buzz", "wrong"
- **Background**: "game music", "background music", "loop", "ambient"

## Lưu ý:
- File nên ở định dạng MP3 hoặc WAV
- Dung lượng < 1MB cho hiệu suất tốt
- Nên chọn âm thanh ngắn (1-3 giây) cho correct/wrong
- Background music nên là loop (có thể lặp lại)
